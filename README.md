# MyAndroidLibrary

Kotlin 移植バージョン

## `widget`モジュール

`library/widget`  

カスタムViewを集めた

- HorizontalListView  
  RecyclerViewを拡張した横方向にスクロールするListViewのようにふるまうwidget
- CustomNumberPicker  
  NumberPickerを拡張して負数も表示・選択できるwidget
- FloatPicker  
  小数を表示・選択できるwidget
- ExpandableTextView  
  表示する文字列の長さ・viewの横幅に合わせて`textScaleX`を自動調節するwidget
